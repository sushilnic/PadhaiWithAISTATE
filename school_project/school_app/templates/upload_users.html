📖 Upload Instructions (English)
  • Required Columns: email, username, password, school_name, nic_code, block_id
  • email – School login ID (must be unique)
  • username – School short code or name (internal use)
  • password – Initial password (school can change later)
  • school_name – Full school name (can include NIC code)
  • nic_code – School NIC/UDISE code
  • block_id – Associated block ID (from Block master table)
  • Important: Column headers must match exactly, otherwise data won't upload
